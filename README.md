# PAシフト自動作成ツール (仮)

## 概要
軽音サークルで副PA長をしているのですが、毎回LINEで希望を集めて手作業でシフトを組むのがパズルみたいで大変でした。そこで、自分の負担を減らすためにこの自動化プログラムを作りました。

## 使用技術
* Python

## 工夫した点・アルゴリズム
* 単純なランダム割り当てではなく、メンバーに「スキルポイント」と「優先度」を持たせました。
* 卓（メイン）とステージ（サポート）でそれぞれ必要な基準点を設け、初心者と経験者が適切にペアになるようアルゴリズムを組んでいます。
* また、来年以降に引き継ぐ人にも仕組みがわかるように作成

## 今後の実装予定
* 完成したシフトのExcel(.xlsx)への自動出力機能
* バンド数などが常に変動するためそれに対応できるようにしておきたい
* スマホからの提出の前の中期目標としてシフト管理者が操作しやすいアプリを作る
* Flaskを用いたWebアプリ化（スマホからの希望シフト提出など）
